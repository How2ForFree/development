This document is strictly for people who are trying to upgrade this theme from a previous version. It will discuss the file changes as well as any notes that are important for updating.

Please take the time to create a backup of your previous version of this theme in case something goes wrong. I do my best to make updating as smooth as possible, but it's always best to be safe!

As always, updating is as easy as just over-writing your existing theme-folder with the newest version... but this log helps people who have made customizations to upload JUST the new files. If you haven't messed with the core template files, chances are that all you need to do is upload the new theme folder (keep a backup of your old one!). This won't change your content (images, pages, posts, etc.) at all.

Version 2.0.X

Version 2.0.X is part of a much larger update of all of my themes (by Brandon R Jones, aka Epicera at ThemeForest). All themes have been moved into a new naming convention, where "2.0" represents the framework version, and the trailing ".X" represents the actual version number of the individual theme. This is part of a larger effort to universalize the theme framework. The major updates are inside the /functions folder, but nearly all of the files in the theme have been editted in accordance with the new framework system. This shouldn't affect your content or anything, so update your theme as you normally would:

  1. Create a backup of your current theme folder inside /wp-content/themes.
  2. Update/overwrite the new folder via FTP.
  3. Clear your browser cache and test the theme out.
  4. Report any bugs to the support forum > http://support.makedesignnotwar.com
Thanks for the continued support and feedback everyone!
